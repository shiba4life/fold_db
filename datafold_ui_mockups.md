## 7. One-Click Sample Loading

The UI will provide a streamlined workflow for loading sample data with a single click:

1. **Sample Library Navigation**:
   - Users can browse categories of samples (schemas, queries, mutations)
   - Each sample has a preview and description

2. **One-Click Loading**:
   - Each sample has a prominent "Load" button
   - Clicking this button immediately loads the sample into the appropriate interface
   - For schemas, the schema is loaded into the database
   - For queries and mutations, they are loaded into the builder interface ready to run

3. **Sample Sets**:
   - Related samples are grouped together
   - Users can load a complete set (e.g., a schema and related queries/mutations) with one click

4. **Quick Access**:
   - "Load Sample" buttons are available in each main interface
   - Dropdown menus provide quick access to relevant samples

This design ensures that users can quickly get started with DataFold by loading sample data with minimal effort.
